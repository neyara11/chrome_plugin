# Dify ChatBot — Chrome расширение

Встраивает чат-ботов [Dify](https://dify.ai/) и [Open WebUI](https://openwebui.com/) на любые веб-страницы через плавающую кнопку.

## Возможности

- **Плавающая кнопка** на сайтах — перетаскивается мышью
- **Drag-to-resize** окно чата
- **Контекстное меню**: отправить выделенный текст или URL страницы в чат
- **Dify**: встраивание через iframe с авто-заполнением переменных
- **Open WebUI**: нативный виджет со стримингом, несколько чатов на сайт, боковая панель
- **Настройка правил сайтов**: разный чат-бот на разных доменах
- **Глобальный режим**: показывать одно приложение на всех сайтах

## Установка

1. Скачай или склонируй репозиторий
2. Открой `chrome://extensions/`, включи **Режим разработчика**
3. «Загрузить распакованное расширение» → выбери папку `src/`
4. Расширение появится в списке

## Настройка

### Dify

1. Создай приложение в Dify, скопируй URL чат-бота (Embed → Chrome Extension)
2. В настройках расширения создай приложение типа **Dify**, вставь URL
3. Настрой переменные для текста (`userinput.query`) и URL (`page_url`)

### Open WebUI

1. Получи API-ключ: Open WebUI → User Settings → Account → API Keys
2. В настройках расширения создай приложение типа **Open WebUI**, укажи:
   - URL инстанса (напр. `https://my-openwebui.com`)
   - API ключ
   - Модель (можно загрузить список кнопкой 🔄)
3. Нажми «Сохранить»

## Использование

- На любом сайте появится цветная кнопка в правом нижнем углу
- Клик — открыть/закрыть чат
- Правый клик на странице → «Dify: отправить URL» / «Dify: отправить текст»
- Иконка расширения — быстрая настройка для текущего сайта
- Страница настроек — управление приложениями и правилами сайтов

## Структура проекта

```
src/
├── manifest.json          # Chrome Extension manifest v3
├── background.js          # Service worker: меню, API-прокси, стриминг
├── content.js             # Content script: кнопка, окно, iframe/виджет
├── lib/
│   ├── storage.js         # chrome.storage, миграции, чаты
│   ├── url-matcher.js     # Сопоставление URL по шаблонам
│   ├── dify-url.js        # Построитель URL для Dify и Open WebUI API
│   ├── owui-widget.js     # Класс виджета Open WebUI (vanilla JS)
│   └── owui-widget.css    # Стили виджета
├── options/               # Страница настроек
│   ├── options.html
│   ├── options.js
│   └── options.css
├── popup/                 # Popup-окно расширения
│   ├── popup.html
│   ├── popup.js
│   └── popup.css
└── icons/                 # Иконки расширения
```
